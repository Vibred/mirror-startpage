# mirror-startpage
这是一个 Startpage 搜素引擎的镜像。搜索结果基本上来自谷歌。
来自中国大陆的用户无法直接访问 Google 和 Startpage。
# 为什么不镜像直接镜像谷歌（Google）？
因为Google频繁检测流量，跳人机验证。
